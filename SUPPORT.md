## Before asking for help

Please check the following first:

1. **Configuration**
   - Did you edit `config.yml` correctly? (YAML syntax matters)
   - Did you run `/cl reload` after changes?

2. **Server requirements**
   - Minecraft version: **1.21 – 1.21.4**
   - Server software: **Paper / Spigot / Purpur** (Vanilla servers do NOT support plugins)
   - Java version: **21+**

3. **Locator Bar (if not working)**
   - Client must be **Minecraft 1.21.6+**
   - Server plugin must have `locator-bar.enabled: true` in config

4. **Mod version (client)**
   - Minecraft: **1.21.6 – 26.1.2**
   - Mod loader: **Fabric** or **Forge**

## How to report a bug

Open a [GitHub issue](https://github.com/kasperoid31/compasslocator/issues/new) with:

- **Plugin/mod version**
- **Minecraft version**
- **Server software** (for plugin) or **mod loader** (for mod)
- **Config.yml** (if relevant)
- **Steps to reproduce**
- **Expected vs actual behavior**
- **Latest.log** or server logs (if any errors)

## Quick answers

| Question | Answer |
|----------|--------|
| Plugin not loading? | Check `plugins/` folder, restart server, check Java version |
| Locator Bar not working? | Client needs Minecraft 1.21.6+ |
| Recovery compass not showing? | You need to die at least once |
| Config not applying? | Run `/cl reload` or restart server |
| Mod not working? | Check Fabric/Forge version, put `.jar` in `mods/` folder |
